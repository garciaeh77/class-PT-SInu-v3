# Validation Data for the SINu Port

This directory contains the reference inputs and outputs used to validate the self-interacting neutrino (SINu) physics port from `class-interacting-neutrinos-PT` (commit acd14cb) into `class_snu_uptodate` (based on `class_public`, commit e858083).

## What Is Here

| Path | Contents |
|------|----------|
| `vanilla_sync.ini` | Input for vanilla ΛCDM run (SINu off), synchronous gauge, Planck 2018 best-fit cosmology |
| `sinu_sync_massless.ini` | Input for SINu run with massless neutrinos (log10\_G\_eff\_nu = -1.5) |
| `sinu_sync_massive.ini` | Input for SINu run with one massive neutrino (m\_ncdm = 0.06 eV) |
| `ref_vanilla_sync/` | Reference outputs from class-interacting-neutrinos-PT, vanilla case |
| `ref_sinu_sync_massless/` | Reference outputs, SINu with massless neutrinos |
| `ref_sinu_sync_massive/` | Reference outputs, SINu with one massive neutrino (0.06 eV) |
| `MANIFEST.md` | Detailed description of each test case and its outputs |

All reference outputs were generated by the `class-interacting-neutrinos-PT` binary and cover background quantities, unlensed and lensed CMB power spectra (TT, EE, TE, BB, phiphi), and the matter power spectrum P(k).

## Test Cosmology

All test cases use the same base cosmology (Planck 2018 best-fit):

| Parameter | Value |
|-----------|-------|
| h | 0.6732 |
| omega\_b | 0.022383 |
| omega\_cdm | 0.12011 |
| A\_s | 2.1005 × 10⁻⁹ |
| n\_s | 0.96605 |
| tau\_reio | 0.0543 |

**Massless case**: N\_ur = 3.044, N\_ncdm = 0  
**Massive case**: N\_ur = 2.0328, N\_ncdm = 1, m\_ncdm = 0.06 eV, T\_ncdm = 0.71611

SINu coupling: **log10\_G\_eff\_nu = -1.5** (i.e., G\_eff\_nu ≈ 0.0316 GeV⁻²)

## Regenerating Reference Outputs

The reference data was generated using `class-interacting-neutrinos-PT`. To regenerate it:

```bash
# 1. Build class-interacting-neutrinos-PT (requires OpenBLAS on macOS/Apple Silicon)
brew install openblas   # if not already installed
cd class-interacting-neutrinos-PT
make clean && make OMPFLAG=""

# 2. Run each test case (from the repository root)
cd class-interacting-neutrinos-PT
./class ../validation_data/vanilla_sync.ini
cp output/vanilla_sync_*.dat ../validation_data/ref_vanilla_sync/

./class ../validation_data/sinu_sync_massless.ini
cp output/sinu_sync_massless_*.dat ../validation_data/ref_sinu_sync_massless/

./class ../validation_data/sinu_sync_massive.ini
cp output/sinu_sync_massive_*.dat ../validation_data/ref_sinu_sync_massive/
```

> Note: `class-interacting-neutrinos-PT` forces `P_k_max_h/Mpc = 100` internally, so P(k)
> reference files extend to ~106 h/Mpc with 622 k-values regardless of the .ini setting.
> The comparison script handles this by interpolating onto a common k-grid.

## Running the Regression Tests

Use `compare_outputs.py` at the repository root. Build `class_snu_uptodate` first:

```bash
cd class_snu_uptodate && make clean && make && cd ..
```

### Vanilla regression (should match to machine precision)

```bash
cd class_snu_uptodate
./class ../tests/vanilla_sync.ini
cd ..
python compare_outputs.py \
  --reference validation_data/ref_vanilla_sync \
  --test class_snu_uptodate/output \
  --bg-tolerance 0.0001 --cl-tolerance 0.001 --pk-tolerance 0.001
```

### SINu validation — massless neutrinos

```bash
cd class_snu_uptodate
./class ../validation_data/sinu_sync_massless.ini
cd ..
python compare_outputs.py \
  --reference validation_data/ref_sinu_sync_massless \
  --test class_snu_uptodate/output \
  --bg-tolerance 0.0001 --cl-tolerance 0.01 --pk-tolerance 0.001
```

### SINu validation — massive neutrinos

```bash
cd class_snu_uptodate
```

## Validation Tolerances and Expected Results

The following tolerances apply. The `class_public` → `class-interacting-neutrinos-PT` version
upgrade introduces an unavoidable baseline difference (~0.4–0.6% in C_l TT/EE) due to the
HyRec2020 vs older hyrec recombination codes:

| Quantity | Tolerance | Measured (massless) | Measured (massive) |
|----------|-----------|---------------------|-------------------|
| Background (H, distances, densities) | 0.01% | < 1×10⁻⁶ | < 1×10⁻⁶ |
| P(k) | 0.1% | 0.021% | 0.021% |
| C_l TT (lensed) | 1.0% | 0.42% | 0.41% |
| C_l EE (lensed) | 1.0% | 0.59% | 0.60% |

The growth factor columns (`gr.fac. D`, `gr.fac. f`) FAIL comparison due to a known
cross-version normalization change and should be ignored.

TE, phiphi, TPhi, and Ephi FAIL due to zero-crossing artifacts in relative difference
computation; these are expected and documented.

## Python Wrapper Validation

To validate the Python (`classy`) interface against CLI:

```bash
python tests/validate_classy_vs_cli.py
```

This compares classy and CLI output to machine precision (< 10⁻¹² relative difference)
for both massless and massive SINu cases.
