# Validation Data Manifest

This directory contains reference inputs (.ini files) and outputs for validating the SINu port from class-interacting-neutrinos-PT to class_snu_uptodate. All reference outputs were generated by the class-interacting-neutrinos-PT binary built from commit acd14cb (see REFERENCE at repository root).

## Test Cases

### vanilla_sync

- **Input**: `vanilla_sync.ini`
- **Output directory**: `ref_vanilla_sync/`
- **Description**: Standard LCDM cosmology with no self-interacting neutrinos (SINu off). Synchronous gauge. Planck 2018 best-fit parameters (h=0.6732, omega_b=0.022383, omega_cdm=0.12011, A_s=2.1005e-9, n_s=0.96605, tau_reio=0.0543). All neutrinos massless (N_ur=3.044, N_ncdm=0).
- **Outputs**: background (4620 rows, 16 columns), cl (2999 multipoles l=2..3000, 8 columns: l, TT, EE, TE, BB, phiphi, TPhi, Ephi), cl_lensed (same format), pk (622 k values, 2 columns: k, P(k)).
- **Purpose**: Establishes the vanilla baseline for the old CLASS version. Comparing this against class_snu_uptodate (= class_public) vanilla output measures the inherent version difference that sets the floor for SINu validation accuracy.

### sinu_sync_massless

- **Input**: `sinu_sync_massless.ini`
- **Output directory**: `ref_sinu_sync_massless/`
- **Description**: SINu enabled with log10_G_eff_nu = -1.5 (moderate self-interaction). Synchronous gauge. All neutrinos massless (N_ur=3.044, N_ncdm=0). Same cosmological parameters as vanilla_sync.
- **Outputs**: background, cl, cl_lensed, pk (same column formats as vanilla).
- **Purpose**: Reference for SINu validation with massless neutrinos. The SINu collision terms modify the massless neutrino Boltzmann hierarchy using precomputed collision integrals from neutrinos_collision_terms/Coll_integrals_5_qbins.dat.

### sinu_sync_massive

- **Input**: `sinu_sync_massive.ini`
- **Output directory**: `ref_sinu_sync_massive/`
- **Description**: SINu enabled with log10_G_eff_nu = -1.5. Synchronous gauge. One massive neutrino species (N_ncdm=1, m_ncdm=0.06 eV, T_ncdm=0.71611) with the remaining neutrinos massless (N_ur=2.0328). Same cosmological parameters as vanilla_sync otherwise.
- **Outputs**: background, cl, cl_lensed, pk (same column formats; background has additional ncdm columns).
- **Purpose**: Reference for SINu validation with massive neutrinos. The SINu collision terms modify both the massless and massive neutrino hierarchies.

## Regeneration

To regenerate the reference data:

1. Build class-interacting-neutrinos-PT: `cd class-interacting-neutrinos-PT && make clean && make OMPFLAG=""` (OpenBLAS required at /opt/homebrew/opt/openblas/lib/libopenblas.dylib).
2. Run each .ini file from the CLASS directory using the root-level input files: `cd class-interacting-neutrinos-PT && ./class ../validation_data/<case>.ini`
3. Copy outputs back to the root-level validation directory: `cp class-interacting-neutrinos-PT/output/<prefix>_*.dat validation_data/ref_<case>/`

## Comparison

Use compare_outputs.py at the repository root:

    python compare_outputs.py \
      --reference validation_data/ref_<case> \
      --test class_snu_uptodate/output \
      --bg-tolerance 0.0001 --cl-tolerance 0.001 --pk-tolerance 0.001
